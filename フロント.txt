🌤【気象情報 API】（天気・風・雨・気温・体感・警報など）
✔ 1) Open-Meteo

特徴：無料・APIキー不要で即使えるグローバル気象 API（風速・降水・気温など取得可）

料金：無料（商用利用も基本無料だが利用条件要確認）

用途：リアルタイム・予報・履歴 → 徒歩・車どちらの気象影響評価に◎

おすすめ度：⭐️⭐️⭐️（まず試すならコレ）

ライセンス：Creative Commons CC BY 4.0
公式URL：https://open-meteo.com/

✔ 2) OpenWeatherMap API

特徴：現在〜予報・履歴・警報まで対応。世界対応。

料金：無料枠あり（1日1000回等）・商用プランへアップ可能

注意：2023–2024以降、One Call 3.0 への移行で無料枠制限あり

用途：「予報＋警報」含め使いたい場合に強い

おすすめ度：⭐️⭐️（無料枠だけで使うなら要設計）

ライセンス：Proprietary（利用規約ベース）
公式URL：https://openweathermap.org/api

✔ 3) 気象庁系 WAGRI API / 気象庁データラッパー

特徴：日本気象庁公式データを API 形式で使える（府県予報・警報・1km メッシュなど）

料金：無料（公的オープンデータ）

用途：国内高精度気象 → 地域特化の調査に◎

おすすめ度：⭐️⭐️（日本内向けでは強い）

ライセンス：政府標準利用規約（第2.0版）
公式URL：https://www.wagri.net/

気象 API のおすすめまとめ
1️⃣ Open-Meteo — とりあえず無料で始めやすい
2️⃣ OpenWeatherMap — 機能豊富 / 無料枠ありだが制限注意
3️⃣ 気象庁系/WAGRI — 日本国内高精度データに最適

🚗【交通・渋滞・交通量データ API】
✔ 1) 国土交通省 交通量 API（オープンデータ）

特徴：全国直轄国道の交通量データ（5分/1時間値）を取得可能（GeoJSON/CSV）

料金：無料（規約同意のみ必要）

用途：渋滞傾向・混雑度解析・簡易判定に◎

おすすめ度：⭐️⭐️⭐️（基礎交通量として最初に）

ライセンス：政府標準利用規約（第2.0版）
公式URL：https://www.mlit.go.jp/road/ir/ir-data/data_shu.html

✔ 2) TomTom Traffic APIs

特徴：リアルタイム渋滞/交通インシデント/滞留データなど豊富な交通情報を提供

料金：無料トライアルあり、有料プランが基本（従量制）

用途：車両ルート最適化・リアルタイム渋滞回避に◎

おすすめ度：⭐️⭐️（精度欲しい場合）

ライセンス：Proprietary（TomTom 利用規約）
公式URL：https://developer.tomtom.com/traffic-api

✔ 3) HERE Real-Time Traffic API

特徴：HERE マップのリアルタイム・過去交通データ提供（プローブ＋センサー情報）

料金：無料枠あり / 有料プランあり（法人向け）

用途：物流/車両向けリアルタイム渋滞・ETA最適化

おすすめ度：⭐️⭐️（Google の代替として多機能）

ライセンス：Proprietary（HERE 利用規約）
公式URL：https://developer.here.com/products/traffic

交通 API のおすすめまとめ
1️⃣ 国交省交通量 API — 費用ゼロで道路流量把握に最適
2️⃣ TomTom Traffic — 商用精度のリアルタイム交通情報
3️⃣ HERE Traffic — 広域路線・ETA 最適化に強い

🧭【人流・歩行者密度関連 API】（徒歩パフォーマンス向け）

※ 日本では無料で簡単に使える“歩行者密度 API” は公的にはほぼ存在しません。多くは 有料プローブデータ系 となります。

✔ 1) Waze for Cities / Connected Citizens Program

特徴：交通・事故・混雑等を自治体向けに共有するインシデント型データ

料金：基本無料（要参加/契約）

用途：都市部のリアルタイム渋滞・歩行者通行状況の傾向推定

おすすめ度：⭐️⭐️

ライセンス：Proprietary（Waze CCP 規約）
公式URL：https://www.waze.com/ccp

✔ 2) SafeGraph / Veraset（プローブ位置情報）

特徴：スマホ位置データを人流/人の密度/滞留情報として取得可能

料金：有料（データ契約ベース）

ライセンス：Proprietary
公式URL：https://www.safegraph.com/

公式URL：https://www.veraset.com/

✔ 3) Google Maps Popular Times / Crowd Prediction

特徴：店舗/場所の混雑予測データ（利用は API 外観 or スクレイピング的アプローチ）

料金：API 直接非公開。Google Maps API 上の付随情報として利用可能

注意：ライセンスに注意が必要

ライセンス：Proprietary（Google Maps Platform 利用規約）
公式URL：https://developers.google.com/maps
